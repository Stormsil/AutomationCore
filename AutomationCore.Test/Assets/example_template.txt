# Папка для шаблонов изображений

Поместите сюда PNG файлы для тестирования распознавания изображений.

## Примеры названий файлов:
- notepad_menu.png - меню блокнота
- save_button.png - кнопка сохранения
- close_button.png - кнопка закрытия
- file_menu.png - меню "Файл"

## Как создать шаблон:
1. Откройте любое приложение
2. Сделайте скриншот (Win + Shift + S)
3. Обрежьте нужный элемент
4. Сохраните как PNG в эту папку
5. Перезапустите демо для тестирования
